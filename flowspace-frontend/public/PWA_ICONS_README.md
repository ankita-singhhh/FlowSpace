# PWA Icons Required

FlowSpace needs PWA icons to be fully installable as a Progressive Web App.

## Required Icons

Add the following icons to this `public/` folder:

1. **icon-192x192.png** (192x192 pixels)
   - Used for Android devices and smaller displays
   - Should have a transparent background
   - Recommended: Simple logo on colored background

2. **icon-512x512.png** (512x512 pixels)
   - Used for larger displays and splash screens
   - Should have a transparent background
   - Recommended: Same design as 192x192 but higher resolution

## Quick Solution

You can generate these icons using:
- **Favicon.io**: https://favicon.io/ - Upload your logo and it generates all sizes
- **Canva**: Create a 512x512 design and export as PNG
- **Online tools**: https://realfavicongenerator.net/

## Design Guidelines

- Use the FlowSpace brand color (#3B82F6 blue)
- Keep it simple and recognizable at small sizes
- Avoid text - icons work better
- Ensure good contrast
- Use transparent background for flexibility

## Temporary Placeholder

If you want to test without icons, the app will still work but won't show the install prompt on mobile devices.
